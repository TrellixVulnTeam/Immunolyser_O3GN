Welcome to Immunolyser 1.0



Immunolyser is an automated web-based tool developed for immunologists to analyse immunopeptidomics data in a seamless pipeline based on the integration of multiple computational tools for peptide analysis. The pipeline generates a comprehensive report summarising the sequence conservation, clustering of similar peptides in groups and prediction of binding affinity to various MHC allotypes based on the uploaded peptide data. The pipeline has a user-friendly interface available online, with no requirement of programming experience.

Please refer help page for detailed guidelines on how to use the pipeline and analyse the web-based output reports.

How to use?

Use Initialiser module to submit a job. Following input will be required to run the analysis.

  • Sample file(s): Sample file is a csv file and should have 'Peptide', 'Length' and 'Accession' columns. Multiple samples can be uploaded and for every sample, multiple replicate files can be uploaded by seleting multiple files.
  • Control file(s) (optional): Control file is a csv file and should have 'Peptide' column. Peptide present in the this file are labelled as 'control' in downloadable results files. User can also upload multiple control files in case control datasets are present across differnt files.
  • Alleles of interest (optional): HLA-A, HLA-B or HLA-C alleles of interest for peptide-MHC binding predictions.
